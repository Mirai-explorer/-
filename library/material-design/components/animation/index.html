动画

运动中的材料反应灵敏而自然。使用这些缓和曲线和持续时间模式来创建平滑且一致的运动。

设计与 API 文档

  • 材料设计指南：运动

安装

 
npm install @material/animation

用法

Sass 变量

我们提供可与 animation 或 transition CSS 属性配合使用的定时功能

​x
 
@use "@material/animation";
​
.my-element--animating {
  animation: foo-keyframe 175ms animation.$standard-curve-timing-function;
}
变量 说明
$deceleration-curve-timing-function 定时功能：减速
$standard-curve-timing-function 定时功能：快速加速后缓慢减速
$acceleration-curve-timing-function 定时功能：加速
$sharp-curve-timing-function 定时功能：快速加速后减速

下面的函数根据 $name 和 $duration 来创建过渡效果。您还可以指定 $delay，但默认值是0ms。 $name 可以引用关键帧，也可以引用 transition 中的 CSS 属性。

 
@use "@material/animation";
​
.my-element {
  transition: animation.exit-permanent(/* $name: */ opacity, /* $duration: */ 175ms, /* $delay: */ 150ms);
  opacity: 0;
  will-change: opacity;
​
  &--animating {
    transition: animation.enter(/* $name: */ opacity, /* $duration: */ 175ms);
    opacity: 1;
  }
}
 
@use "@material/animation";
​
@keyframes fade-in {
  from {
    transform: translateY(-80px);
    opacity: 0;
  }
​
  to {
    transform: translateY(0);
    opacity: 1;
  }
}
​
.my-element {
  animation: animation.enter(/* $name: */ fade-in, /* $duration: */ 350ms);
}
函数 说明
enter($name, $duration, $delay) 定义进入帧的过渡
exit-permanent($name, $duration, $delay) 定义彻底退出帧的过渡
exit-temporary($name, $duration, $delay) 定义临时退出帧的过渡

JavaScript

这些函数处理用于各种浏览器的前缀

 
import {getCorrectEventName} from '@material/animation';
​
const eventToListenFor = getCorrectEventName(window, 'animationstart');
方法声明 说明
getCorrectEventName(windowObj: Window, eventType: StandardJsEventType) => StandardJsEventType \| PrefixedJsEventType 返回 JavaScript 事件名，必要时加上前缀。关于受支持的值，请参阅 types.ts
getCorrectPropertyName(windowObj: Window, cssProperty: StandardCssPropertyName) => StandardCssPropertyName \| PrefixedCssPropertyName 返回 CSS 属性名，必要时加上前缀。关于受支持的值，请参阅 types.ts
